# Crop Yield Prediction Using Machine Learning

This project predicts crop yield using machine learning and rainfall data.

## Features
- State selection
- Crop selection
- Season selection
- Area input
- Cultivation year input
- Automatic rainfall estimation
- XGBoost prediction model

## Model Performance
- R² Score: 0.8795
- MAE: 11.92

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- XGBoost
